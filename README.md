# Resume Screener - AI-powered resume to JD matching using NLP# Resume Screener

# 

# AI-powered Resume Screener that matches resumes with job descriptions using NLP (Sentence-Transformers) and predicts a match score. Includes FastAPI backend and Streamlit dashboard.

# 

# \## Datasets

# Download from Kaggle and place in `data/raw/`:

# \- Resume Dataset (categories): https://www.kaggle.com/datasets/snehaanbhawal/resume-dataset

# \- Recruitment Dataset (resume + JD + match label): https://www.kaggle.com/datasets/surendra365/recruitement-dataset
